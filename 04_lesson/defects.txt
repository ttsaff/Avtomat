BUG-001
Название: Методы не обрабатывают None
Описание:
При передаче None в качестве параметра string методы класса StringUtils
выбрасывают AttributeError.
Ожидаемое поведение:
Методы должны либо выбрасывать понятное исключение (ValueError),
либо корректно обрабатывать None.
Фактическое поведение:
AttributeError: 'NoneType' object has no attribute ...
Приоритет: Средний

BUG-002
Название: Метод contains возвращает True при пустом symbol
Описание:
Метод contains("SkyPro", "") возвращает True.
Это поведение может быть неожиданным для пользователя библиотеки.
Ожидаемое поведение:
Вернуть False или выбросить исключение.
Фактическое поведение:
Возвращается True.
Приоритет: Низкий
